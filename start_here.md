Für den Zugang zu unserer Umgebung wird ein gültiger Zugangs-Token benötigt.
Dieser kann initial über 

HIER REQUEST-BEISPIEL

Abruf mit cURL:
```curl
curl -X POST \
  https://api.otto.market/v1/token \
  -H 'Content-Type: application/x-www-form-urlencoded' \
  -H 'cache-control: no-cache' \
  -d 'username=<YourUsername>&password=<YourPassword>&grant_type=password&client_id=token-otto-api'
```
